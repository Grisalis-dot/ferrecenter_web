# FerreCenter 🛠️

Plataforma de comercio electrónico completa para ferretería, desarrollada con el stack MERN (MongoDB, Express, React, Node.js).

## Características Principales

- **Catálogo de Productos:** Búsqueda, filtrado por categorías y gestión de inventario.
- **Carrito de Compras:** Gestión de estado global y persistencia.
- **Pasarela de Pagos:** Integración con Wompi (Bancolombia) para pagos reales.
- **Panel Administrativo:** Gestión de productos, usuarios, pedidos y configuración de la empresa.
- **Facturación:** Generación automática de facturas PDF y envío por correo electrónico.
- **Seguridad:** Autenticación JWT y protección de rutas.

## Estructura del Proyecto

- `frontend/`: Aplicación cliente construida con React y Vite.
- `backend/`: API RESTful construida con Node.js y Express.

## Despliegue

- **Frontend:** Vercel
- **Backend:** Render
- **Base de Datos:** MongoDB Atlas
